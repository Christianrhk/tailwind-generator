So funktionieren Online Buchungen


1. Corona Schnelltest online buchen

Um einen Corona-Schnelltest zu buchen, reservieren Sie einfach den gewünschten Termin online. Im Anschluss erhalten Sie eine Terminbestätigung per E-Mail.

Bitte beachten Sie: NUR symptomfreie Personen dürfen bei uns getestet werden. Sollten Sie Husten, Fieber, oder andere grippeähnliche Symptome haben, wenden Sie sich bitte an einen Hausarzt.

2. Termin wahrnehmen

Um Wartezeiten zu vermeiden, bitten wir Sie, pünktlich zum reservierten Termin zu erscheinen und der Beschilderung vor Ort zu folgen. Bitte beachten Sie die Regelung zur Maskenpflicht. Auf unserem gesamten Gelände ist das Tragen einer medizinischen Maske vorgeschrieben.

3. Empfang

Am Empfang erhalten Sie gegen Vorlage Ihres Personalausweises ein Testblatt mit individuellem QR-Code zur späteren Ergebnisabfrage über das Smartphone oder das Internet.

4. Test

An der Teststation entnimmt unser medizinisch geschultes Personal den Abstrich. Danach können Sie das Testzentrum direkt verlassen. (Dauer ca. 3 Minuten)

5. Testergebnis

Nach ca. 15-30 Minuten wird Ihr Testergebnis ausgewertet und ist für Sie über den individuellen QR-Code einsehbar.

Falls sie einen PCR-Test gebucht haben, können Sie das Ergebnis innerhalb von ca. 24-72 Stunden abrufen.
